# VoxTubeS Listening Demo

This repository contains a small static listening-demo page for the VoxTubeS paper. It includes 5 randomly selected speakers, 3 utterances per speaker, and aligned authentic/synthetic examples for the evaluated methods.

The site is intended to be published with GitHub Pages from the repository root.

## Files

- `index.md`: GitHub Pages landing page.
- `audio/`: demo audio files referenced by the page.
- `selection_manifest.csv`: speaker and utterance selection manifest.

## GitHub Pages

After pushing this repository to GitHub, enable Pages from:

`Settings` -> `Pages` -> `Build and deployment` -> `Deploy from a branch`

Select:

- Branch: `main`
- Folder: `/ (root)`

The page should then appear at:

`https://<github-user>.github.io/<repo-name>/`
